Agent Results

Table-state parsing remains a bottleneck for embodied agents.

Strict Overall

Overall perception is strict problem-level exact match: every applicable field must be correct for a problem to count as solved.

Reliable Coarse Fields

Blind and turn fields are easier than chip and bet fields, but complete table-state parsing remains far from solved.

System Impact

Incorrect chip, bet, or turn parsing can select the wrong high-level action before any low-level policy failure occurs.

System-Level Evaluation

Closed-loop hands expose accumulated waits, recoveries, and primitive dispatches.

The system-level protocol pairs a GPT 5.5 agent harness with the pi0 dexterous policy. Each rollout repeatedly captures state, routes through wait and verification gates, and dispatches card or chip primitives only when physical motion is required.
