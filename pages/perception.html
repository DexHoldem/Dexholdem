Perception Problem

Agentic Perception Bench

A controlled state-parsing benchmark that asks an embodied agent to recover the routing-relevant tabletop state before it selects poker actions or robot primitives.

Task Definition

Parse visual state into auditable routing facts.

36 Fixed Problems

The released problem list covers turn gates, opponent-wait states, robot actions in progress, held-card reads, cached-sequence continuation, recovery cases, table decisions, fold wins, showdown outcomes, and collect-winnings sequences.

Required Artifacts

Each run writes visual_summary.json, eval_report.md, and nonempty visual evidence under visual_raw/. Missing or invalid artifacts are scored as incorrect for the affected fields.

Deterministic Scoring

The evaluator normalizes the submitted summary and compares it against hidden labels with strict problem-level exact match plus field-wise diagnostic accuracy.
